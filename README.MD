## Hosted Project

[E-Commerce API Heroku URL](https://e-commerce-api-10.herokuapp.com/)

- A Simple E-Commerce API Project that lets user to do all the CRUD functionality for example:=

 *Authentication*-
    1.Login User
    2.Logout User
    3.Register User



  *Order*-
    1.Create Order
    2.Get All Orders
    3.Get Single Order
    4.Show Current User Orders
    5.Update Order



*Product*-
     1.Create Product
     2.Delete Product
     3.Get All Products
     4.Get Single Product
     5.Get Single Product Reviews
     6.Update Product
     7.Upload Image



*Review*-
  1.Create Review
  2.Delete Review
  3.Get All Reviews
  4.Get Single Review
  5.Update Review


*User*-
  1.Get All Users 
  2.Get Single User
  3.Show Current User
  4.Update User
  5.Update User Password
